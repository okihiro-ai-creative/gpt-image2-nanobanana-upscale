# gpt-image2-nanobanana-upscale

Created by OKIHIRO — AI自動化ワークフローを130個以上開発・運用しています。

`gpt-image2-nanobanana-upscale` は、GPT Image 2で生成した完成画像をNano Banana Proで崩さずアップスケール・クリーンアップするためのプロンプトを、AIエージェントに判断・作成させるスキルです。

## 何ができるか

- 画像の文字の小ささ・潰れ具合・誤字許容度から、文字厳密固定版と短縮版のどちらを使うべきかを判定
- 文字厳密固定版が必要な場合、画像内の全文字をそのまま書き出してプロンプトに反映
- 構図・人物・文字・配色を維持したまま、ノイズ除去・高精細化・文字可読性向上を行うプロンプトの構成
- 手や顔、発光エフェクトなど崩れやすい箇所への補足指示の追加

## 注意

- GPT Image 2とNano Banana Proの利用環境は利用者自身で用意してください。
- 外部サービスのアカウントや利用料金は利用者側の管理です。
- このスキルは再デザインではなく、既存画像の品質改善用です。
